### 🐹 adieu 소개
##### 'adieu'는 반려동물을 입양받고 싶은 사람이나 분양하고 싶은 사람들이 쉽게 반려동물을 입양/분양할 수 있도록 도와주는 프로그램입니다.
### 🐸 실행 순서
1. 'start.py' 실행
2. 로그인 / 회원가입 하기
3. 게시물 둘러보기 혹은 게시물 등록하기 등 원하는 메뉴 실행하기
4. 실행 완료 시 로그아웃 클릭  
   
### 🐶 주요 기능
   - 🐾 회원가입/로그인
   - 🐾 분양 희망하는 반려동물 등록
   - 🐾 메인화면에서 반려동물들 클릭해서 보기
   - 🐾 해당 반려동물 신청
   - 🐾 종류별로 검색하기
   - 🐾 올린 게시물, 신청한 게시물 보기
   - 🐾 사용자 정보, 올린 게시물 정보 수정 가능
   - 🐾 신청한 사용자들 정보 확인 후 수락/거절
